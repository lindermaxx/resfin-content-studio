## Story

**Como** Max Linder,
**Quero** que o app busque automaticamente trending topics de finanças e medicina via Gemini API,
**Para que** eu tenha sugestões reais de temas relevantes para o @residenciaemfinancas sem precisar pesquisar manualmente.

---

## Acceptance Criteria

- [ ] AC1: `POST /api/research/trends` retorna 10 trending topics em JSON
- [ ] AC2: Cada topic contém: `titulo`, `plataforma`, `pilar`, `hook`
- [ ] AC3: A busca usa Gemini com Google Search grounding focado em finanças pessoais e médicos brasileiros
- [ ] AC4: Pilar é um dos 7 do brief.yaml: `educacao_financeira`, `investimentos_carteira`, `organizacao_orcamento`, `mentalidade_comportamento`, `noticias_comentadas`, `solucoes_patrimoniais`, `bastidores`
- [ ] AC5: Resposta em menos de 15 segundos (timeout configurado)
- [ ] AC6: Em caso de erro da API, retorna mensagem descritiva em português

---

## Scope

**IN:**
- API route Next.js em `src/app/api/research/trends/route.ts`
- Integração com `@google/generative-ai` (já instalado)
- Prompt estruturado para retornar JSON com os 10 topics
- Tratamento de erro com mensagem em português

**OUT:**
- UI (é a Story 2.2)
- Cache de resultados
- Histórico de pesquisas anteriores

## Dev Notes

- Usar `GoogleGenerativeAI` do `@google/generative-ai`
- Modelo: `process.env.GOOGLE_TEXT_MODEL` (`gemini-3-pro-preview`)
- O prompt deve instruir o Gemini a retornar **somente JSON válido** sem markdown
- Estrutura esperada do retorno:
```json
[
  {
    "titulo": "string",
    "plataforma": "Instagram | YouTube | TikTok | LinkedIn | Twitter",
    "pilar": "educacao_financeira | investimentos_carteira | ...",
    "hook": "string (1 frase de abertura)"
  }
]
```
- Foco do prompt: médicos brasileiros, finanças pessoais, planejamento patrimonial, notícias econômicas recentes
