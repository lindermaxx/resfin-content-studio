## Story

**Como** sistema,  
**Quero** persistir os posts gerados com status e metadados no Supabase,  
**Para que** o Pipeline opere com dados reais e histórico.

---

## Acceptance Criteria

- [x] AC1: `POST /api/posts` cria post com status inicial
- [x] AC2: `GET /api/posts` retorna posts ordenados por data
- [x] AC3: `PATCH /api/posts/:id` atualiza campos editáveis
- [x] AC4: `PATCH /api/posts/:id/status` muda status com validação
- [x] AC5: Alterações de status registram evento em log de atividade
