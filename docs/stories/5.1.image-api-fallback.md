## Story

**Como** sistema,  
**Quero** gerar imagens a partir da copy aprovada com fallback de provedor,  
**Para que** o fluxo não pare quando o provedor principal falhar.

---

## Acceptance Criteria

- [x] AC1: `POST /api/image/generate` recebe prompt base + instrução opcional
- [x] AC2: Endpoint retorna 2-4 variações de imagem com URL
- [x] AC3: Falha do provedor principal aciona fallback automático
- [x] AC4: Erros retornam mensagem descritiva em português
