## Story

**Como** sistema,  
**Quero** gerar imagens a partir da copy aprovada com fallback de provedor,  
**Para que** o fluxo não pare quando o provedor principal falhar.

---

## Acceptance Criteria

- [ ] AC1: `POST /api/image/generate` recebe prompt base + instrução opcional
- [ ] AC2: Endpoint retorna 2-4 variações de imagem com URL
- [ ] AC3: Falha do provedor principal aciona fallback automático
- [ ] AC4: Erros retornam mensagem descritiva em português

---

## Tasks

- [ ] 1. Definir contrato de request/response da API
- [ ] 2. Implementar chamada ao provedor principal
- [ ] 3. Implementar fallback para OpenAI
- [ ] 4. Normalizar resposta para a UI
- [ ] 5. Tratar timeout e falhas de autenticação

---

## Definition of Done

- [ ] API retorna variações válidas
- [ ] Fallback funcional
- [ ] `npm run lint` sem erros
